# SimplrOS

SimplrOS is a lightweight, x86-64 Operating System built from scratch. This project is designed for learning and is completely Open Source.

## Stack

- Bootloader: A custom UEFI Application
- Kernel: Written in C with some help from Assembly
- Filesystem: Custom made by me, docs will be added later
- Shell: Also Custom
- Runtime: CPython 3.14 (Statically Compiled)
